Localizator3 — мультиязычность для MODX Revolution 3

Один ресурс, несколько языковых версий полей и TV. Данные в localizator3_content; ID ресурса общий для Fenom, pdoTools и miniShop3. Подходит мультидоменным сайтам на MODX 3.

Менеджер: Vue 3 + PrimeVue 4 через VueTools ≥1.1.2-pl, страница «Языки», вкладка «Локализация» с inline-формой и автопереводом. Витрина: 6 сниппетов, Fenom, hreflang, canonical, sitemap.

Подходит, если нужно:
  • переводить pagetitle, content, SEO и TV без отдельных ресурсов в context;
  • вывести переключатель (getLanguageList) и URL по HTTP host;
  • собрать каталог через pdoResources / pdoMenu / msProducts (getLocalizedResources);
  • локализовать товары и опции miniShop3;
  • индексировать переводы в mSearch;
  • переводить пакетно: Yandex, Google, DeepL, LibreTranslate, MyMemory, CLI.
Сниппеты
  • getLocalizedResources — pdoResources, pdoMenu, msProducts / ms3Products + JOIN локализации
  • getLanguageList — list / dropdown, rank, activeClass
  • makeLocalizedUrl — URL по языку и http_host
  • getLocalizedField — поле или TV на языке
  • getLocalizedCanonical — canonical + hreflang
  • getLocalizedSitemap — sitemap с alternate
Fenom
  • {$pagetitle | locfield}
  • {$option_id | locoptioncaption}
  • {$id | locproductoptionvalue}
Менеджер
  • Localizator3 → Языки — key, http_host, cultureKey, rank
  • Ресурс → Локализация — dropdown, Document / TV, Save, Translate
  • Поля формы: OnBuildLocalizationTabs
  • Vue entry ~14 KB через Import Map VueTools
miniShop3
  • Вкладка на msProduct, locOption, locProductOption
  • getLocalizedResources + msProducts / ms3Products
mSearch
  • Поля {key}-pagetitle, {key}-content
  • fields в mSearchForm по активным языкам
Маршрутизация и SEO
  • HTTP host, cookie localizator3_key, Accept-Language
  • localizator3_404_if_no_localization
  • hreflang, canonical, sitemap
Установка
  1. VueTools ≥1.1.2-pl + Localizator3
  2. Localizator3 → Языки, укажите HTTP host
  3. TV: «TV is available in localizations»
  4. pdoFetch.class = pdotools.pdofetchlocalizator3
  5. Шаблон: [[!getLanguageList]], [[!getLocalizedCanonical]]

Fenom:

{'!getLanguageList' | snippet}
{'!getLocalizedResources' | snippet : [
  'snippet' => 'pdoResources',
  'parents' => 0,

{'!getLocalizedCanonical' | snippet}
Требования
  • MODX 3.0+, PHP 8.2+
  • VueTools ≥1.1.2-pl, pdoTools
  • miniShop3, mSearch по желанию

Overlay-модель: один ресурс, N локализаций. Разная структура по языкам: Babel. MODX 2: Localizator.

События: OnBuildLocalizationTabs, OnSaveLocalization, OnToggleLocalizatorLanguage, OnHasLocalizatorError404.
