# Превью для ModStore и modx.pro

HTML и Markdown для карточки **Localizator3** на [modstore.pro](https://modstore.pro) и поста на [modx.pro](https://modx.pro).

Версия пакета: **1.0.8-beta**.

## Файлы

| Файл | Назначение |
|------|------------|
| **modstore-description.html** | Описание пакета на modstore.pro |
| **modxpro-post.html** | Анонс на modx.pro (HTML-режим редактора) |
| **DESCRIPTION.md** | Полное описание для GitHub |
| **logo-256.png** | 256×256, иконка ModStore (добавить вручную) |
| **banner-1200x630.png** | Баннер страницы пакета (добавить вручную) |

## Использование

- **ModStore:** вставьте `modstore-description.html` в поле описания, проверьте предпросмотр.
- **modx.pro:** вставьте `modxpro-post.html` целиком или по блокам.
- **GitHub:** `DESCRIPTION.md` или корневой `README.md`.

`_preview/` не попадает в transport ZIP.
